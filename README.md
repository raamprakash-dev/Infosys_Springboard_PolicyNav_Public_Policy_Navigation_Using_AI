#  PolicyNav – Public Policy Navigation System

**PolicyNav** is an AI-powered platform designed to simplify how users explore, understand, and interact with complex government policies and public documents. The system leverages modern **Natural Language Processing (NLP)** and **Retrieval-Augmented Generation (RAG)** techniques to allow users to ask questions about public policies and receive accurate, context-based responses derived directly from official policy documents.

The platform provides an intuitive interface that enables users to search, analyze, and understand policy information efficiently, making it useful for students, researchers, and the general public.

---

##  Key Features

- **AI-Powered Policy Question Answering**  
  Users can ask natural language questions about policies and receive AI-generated answers based on relevant policy documents.

- **Retrieval-Augmented Generation (RAG)**  
  Implements a RAG pipeline with a **FAISS vector database** to perform semantic search and retrieve the most relevant policy content before generating responses.

- **Multilingual Support**  
  Supports multiple Indian languages including **English, Hindi, Tamil, Telugu, Kannada, Marathi, and Bengali**, enabling broader accessibility.

- **Policy Document Upload & Processing**  
  Users can upload policy documents (PDFs), which are automatically processed and indexed for intelligent search.

- **AI-Based Policy Summarization**  
  Generates concise summaries of long policy documents for quicker understanding.

- **Policy Readability Analysis**  
  Evaluates policy documents using linguistic metrics to measure complexity and readability.

- **Knowledge Graph Visualization**  
  Interactive visualization showing relationships between policies, organizations, and key entities.

- **Semantic Policy Search**  
  Allows users to search policies based on meaning rather than simple keyword matching.

- **User Authentication System**  
  Secure login and signup functionality with protected user sessions.

- **Activity History Tracking**  
  Maintains a history of user interactions and queries for easier navigation and review.

- **Feedback Collection System**  
  Allows users to submit feedback to improve the platform.

- **Modern Streamlit Dashboard**  
  A clean and interactive user interface built with **Streamlit** for smooth navigation and usability.
